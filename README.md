# Greylock Cheat Menu

Personal project made to learn more about the Rust programming language and game reversing.

## Features

- Basic in-game cheat menu
- DLL injection based
- Written in Rust
- Uses `hudhook`
- Uses `ImGui`
- DirectX 12 rendering

## Installation

1. Download the `.dll` file from the releases section.
2. Inject the DLL into `Greylock-Win64-Shipping.exe` using your preferred injector.

## Tech Stack

- Rust
- hudhook
- ImGui
- DirectX 12
